## 这个文件夹为打包文件夹

> `debug`文件夹为当执行`build.py`并选择`0`时打包的路径，以此方法打包出来的exe文件执行时显示控制台''黑窗口''并保留所有控制台信息
>
> `release`文件夹为当执行`build.py`并选择`1`时打包的路径，以此方法打包出来的exe文件执行时不显示控制台以及控制台信息，适合发给其他人
>
> `publish`文件夹是最终发布文件夹，将此文件夹压缩并发给其他人，他人解压后即可直接使用
